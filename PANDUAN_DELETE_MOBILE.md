## ğŸ†˜ Troubleshooting

### Menu tidak muncul di HP?
**Solusi:**
1. Refresh halaman (tekan F5)
2. Pastikan browser up-to-date
3. Bersihkan cache browser

### Tombol hapus tidak bekerja?
**Solusi:**
1. Pastikan internet terhubung
2. Refresh halaman
3. Cek koneksi ke server

### Form tidak bisa diisi di HP?
**Solusi:**
1. Pastikan keyboard muncul
2. Tap pada field input
3. Gunakan browser Chrome untuk hasil terbaik

### Riwayat tidak muncul?
**Solusi:**
1. Tunggu sebentar (loading)
2. Refresh halaman
3. Pastikan laporan sudah dikirim sebelumnya

### Tampilan berantakan di HP?
**Solusi:**
1. Rotate layar (potrait â†” landscape)
2. Zoom out (pinch outward)
3. Gunakan browser terbaru
